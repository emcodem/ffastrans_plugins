Delete Files Processor
This Processor deletes Files and or folders. If you provide a Folder on input, it will attemt to list all files and folders in this directory. If you need to specify include and exclude criteria, please use the Files Find Processor and specify it's output variable as File or Base Folder.

File or base folder
Possible inputs: A folder, A file, A list of files as JSON array, e.g. from Files Find output

Delete Folders:
Allow deletion of folders
Delete Base Folder:
Also attempts to delete the specified Base Folder thats given in "File or base folder" Input

Recurse:
If a folder is specified on input, recurse into subdirectories

Ignore Errors:
Always report success in the workflow

Test Mode:
Don't delete anything, just fill the Output variables with the list of files and folders that would be deleted
